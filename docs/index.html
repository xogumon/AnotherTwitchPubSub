AnotherTwitchPubSub

This is a library to help you with Twitch PubSub (for Web Browser only)

Installation

Install the library downloading the file from Github:

twitch.pubsub.min.js

Or use a CDN to load the library: https://cdn.jsdelivr.net/gh/xogumon/AnotherTwitchPubSub/dist/twitch.pubsub.min.js
